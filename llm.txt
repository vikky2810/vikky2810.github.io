# Vikram Kamble

> Full Stack Developer and B.Tech Computer Science graduate based in India. This is my personal portfolio and technical blog, covering full-stack web development, Python, and building applications on top of LLM APIs.

I build full-stack web applications — React and Next.js on the frontend, Node.js/Express and Flask on the backend, with MySQL and MongoDB for storage. Several of my recent projects are AI-powered tools built on the Gemini API. I currently work as a Software Engineer Intern at RegiKart Private Limited, and have been freelancing since 2023.

The site is a static site hosted on GitHub Pages. Blog posts are stored as HTML strings in a single JSON file (`/src/blogs.json`) and rendered client-side, so the JSON is the most reliable source for full post content.

## Projects

- [Health-Aware Recipe Modifier](https://github.com/vikky2810/Recipe-Modifier): Full-stack Flask app that generates recipes tailored to user-specified medical conditions using the Gemini API — flags harmful ingredients, recommends safe substitutes, and generates PDF summary reports. [Live demo](https://recipe-modifier-app.vercel.app).
- [AI Explains Repo](https://github.com/vikky2810/ai-explains-repo): Full-stack Next.js app that ingests GitHub repository data and generates AI-powered reports on code quality, architecture, security risks, and performance via the Gemini API, with authenticated session history. [Live demo](https://ai-explains-repo.vercel.app/).
- [Online Python Code Editor](https://github.com/vikky2810/OnlineCodeEditor): Browser-based Python editor built with Flask and CodeMirror that executes user code server-side and streams output back in real time, with syntax highlighting, error handling, and a responsive dark UI. [Live demo](https://viks-online-code-editor.vercel.app/).
